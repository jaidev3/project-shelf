# Project Shelf

## Overview

A React TypeScript project built with Vite.

## Prerequisites

- Node.js (v20+)
- npm

## Setup

1. Clone the repository
2. Install dependencies:
   ```
   npm install
   ```

## Development

- Run development server:
  ```
  npm run dev
  ```

## Build

- Build for production:
  ```
  npm run build
  ```

## Linting

- Run linter:
  ```
  npm run lint
  ```

## Technologies

- React 19
- TypeScript
- Vite
- Tailwind CSS
- Firebase
